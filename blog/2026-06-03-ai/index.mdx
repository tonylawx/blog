<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">最近标普和纳指看起来都很强，但问题是，指数强，不等于市场强。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这也是为什么很多人会有一种割裂感：看指数，像牛市；看自己持仓，像没涨。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">现在美股已经不是传统意义上的指数市场。它变成了一个由 <strong>AI</strong> 龙头带动的选股市场。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;"><strong>NVDA</strong>、<strong>AMD</strong>、<strong>MRVL</strong>、<strong>AVGO</strong> 这些股票，才是真正牵动资金情绪的核心。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;"><strong>MRVL</strong> 因为黄仁勋一句“可能成为下一个万亿市值公司”，股价直接暴拉，盘后继续上涨。这一下把半导体情绪重新点燃。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">前几天软件股很强，<strong>IGV</strong> 被大量资金追捧。看涨期权成交量创纪录，散户流入也非常夸张。</p>
