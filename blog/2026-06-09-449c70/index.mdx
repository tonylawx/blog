</div>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">今天的美股不是一根漂亮阳线那么简单。道指独跌，纳指走强，费城半导体指数收复了上周五约三分之二的跌幅，看上去像是科技股重新站了起来。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但真正的问题在尾盘。指数冲高回落，最后一个小时抛压再次出现，说明市场并没有把上周五的恐慌完全消化。更关键的是，纳指上涨的广度并不好，成分股涨跌接近各半，超过一半涨幅来自少数股票。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">所以今天不能简单理解成牛市回来了。它更像一次<strong>反弹测试</strong>：如果能收复上周五高点，短线恐慌解除；如果跌破上周五低点，抛压还没走完。现在还在中间地带，急着下结论反而危险。</p>
</section>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">本周第一道关口是 <strong>CPI</strong>。虽然 6 月议息会议大概率不会被一份数据完全改写，但 <strong>CPI</strong> 会影响未来几个月的利率定价。现在市场关注的不是美联储嘴上怎么说，而是货币市场拿真金白银怎么押。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">纽约联储消费者调查里，一年期通胀预期小幅降到 3.5%，但三年期和五年期仍分别维持在 3.1% 和 3.0%。通胀预期没有明显松动，就业预期反而变弱。失业后找到新工作的预期概率降到 43.7%，未来一年失业概率升到 15.1%。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这是一种很尴尬的组合：劳动力市场预期变差，但通胀还没真正下来。对成长股来说，最怕的就是这种环境。<strong>利率压力不解除，高估值科技股就很难彻底放松。</strong></p>

<span style="color:#2763e9;font-size:12px;font-weight:800;letter-spacing:0.14em;white-space:nowrap;">2 · 谨慎</span>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">上周五纳指 100 大跌 4.77%。这种跌幅如果发生在周五，历史上并不常见。过去 30 年里，纳指 100 在周五跌 4% 以上，一共只出现过 22 次。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">前 21 次里，有 17 次在下一个周一跌破了周五盘中低点。今天没有跌破，说明这次短线确实比多数历史样本强一点。但更重要的是后面五个交易日。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">历史上 21 次里，有 19 次在随后的周五收盘前跌破了前一个周五低点。也就是说，<strong>大跌之后马上 V 回去，不是常态。</strong>统计不是圣旨，但它能提醒你：这种级别的抛售，通常需要时间消化。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">现在投行的观点也很分裂。摩根士丹利和花旗偏乐观，前者看到标普年底 8000 点，后者把年底目标上调到 8100 点。它们的核心逻辑都是盈利预期上调，尤其是半导体、存储和 <strong>AI</strong> 相关公司的盈利修正。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但看多不代表短线安全。美国银行就更谨慎，维持标普年底 7100 点目标，认为当前还有下跌空间。它们担心高市盈率股票远远跑赢低市盈率股票，这是典型的过度投机信号。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">摩根大通的观点更接近短线防御：<strong>AI</strong> 科技、媒体和电信板块可能还有更多抛售，其他板块的支撑不足以推动指数继续走高。总结一下就是：<strong>长期故事还在，但短期危险信号太多。</strong></p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">今年的 <strong>IPO</strong> 风险不能只看 <strong>SpaceX</strong>。<strong>SpaceX</strong> 预计本周定价和上市，后面还有 <strong>OpenAI</strong>、<strong>Anthropic</strong>、<strong>Databricks</strong>、<strong>Stripe</strong> 等大型公司可能在下半年陆续登场。几家加起来，潜在市值和融资关注度都非常夸张。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">历史经验很清楚：大型 <strong>IPO</strong> 潮，通常出现在市场估值高、买家情绪旺、资金愿意付高价的时候。2000 年互联网泡沫，2020 到 2021 年 <strong>SPAC</strong> 和 <strong>IPO</strong> 狂潮，都是类似的情绪高点。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;"><strong>IPO</strong> 潮不一定立刻砸崩市场，但它一定是风险温度计。<strong>SpaceX</strong> 的冲击也不是一天结束，它还有 60 天、180 天等后续解禁节奏。再叠加下半年其他大型 <strong>IPO</strong>，这不是一个交易日的风险，而是一张拉到年底的流动性时间表。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">接下来最简单的判断方法，就是盯住上周五的高低点。突破上周五高点，短线恐慌解除；跌破上周五低点，回撤继续加深；如果一直横在中间，那就是多空都在等催化剂。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">半导体要重点看费城半导体指数。今天它暂时守住趋势线，没有确认破位。但如果后面再次跌破，半导体修复逻辑会重新承压。个股层面，特斯拉已经破趋势线，今天上涨也只能先按反弹看；英伟达和部分半导体权重还在撑盘，但不能替整个市场背书。</p>
