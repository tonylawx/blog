<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">期权，本质上是一份“选择权合约”。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">它给你一个权利：在未来某个日期之前，按照约定好的价格，买入或者卖出一只股票。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">注意，是权利，不是义务。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你可以执行，也可以不执行。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">一张美股期权合约，通常对应 100 股股票。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">Call 可以理解成房产定金。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你看中一套房子，觉得以后会涨，但现在不想一次性买下。你先付定金，锁住未来按今天价格买入的权利。涨了，你有机会赚钱；跌了，你最多亏掉定金。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">Put 更像保险。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你花钱买一个可能性，错了最多亏保费。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">钱马上到账。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这太符合人性了。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">它是你承担风险的价格。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果你不知道自己到底承保了什么灾难，那这笔钱迟早要还回去。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">还有一个很重要的希腊字母，叫 Theta。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">Theta 代表时间价值每天损耗多少。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">对卖方来说，Theta 是朋友。只要市场不剧烈运动，时间每天都在帮你收租。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但注意，Theta 只是朋友，不是亲爹。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">我当年爆仓之前，也觉得 Theta 无敌。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">后来才知道，Theta 每天给你的那点钱，可能只是市场提前支付给你的灾难保费。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你收了保费，就要准备承保灾难。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">3普通人最该先学卖方逻辑 ------------</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">我不建议新手一上来就买短期期权。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">买短期期权需要同时判断方向和时间。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">方向对了，时间错了，照样亏。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">真正更适合普通人慢慢理解的，是两种卖方策略：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">Covered Call，和 Cash Secured Put。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">先说 Covered Call。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你手里已经有 100 股股票，然后卖出一张 Call。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果股票没涨过行权价，你收下权利金。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果股票涨过行权价，你可能按约定价格卖出股票。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">它的本质是：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">用一部分未来上涨空间，换今天确定的现金流。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">我身边有很多朋友，公司分了大量已经归属、可以自由交易的股票，成本很低，甚至接近于“白拿”。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但他们只是握着。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">看着它天天跌。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果他们懂 Covered Call，这些股票就不只是躺在账户里的数字，而是可以每个月产生一点现金流。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">再说 Cash Secured Put。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你本来就想买一只好股票，只是觉得现在价格贵。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">那你可以在自己愿意买入的位置卖 Put。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果股价没跌到，你收下权利金。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果股价跌到了，你按原本愿意接受的价格买入。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">它的本质是：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">收钱等待买入。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这两种策略都不是稳赚。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但它们至少有一个共同点：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你知道自己最坏情况下要做什么。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">Covered Call 最坏是股票被卖掉。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">Cash Secured Put 最坏是你用准备好的现金接股。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这和裸卖 Call 完全不是一回事。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">裸卖 Call 没有保护，风险可以非常大。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">我的三十万，就是刻在这句话下面的学费。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">4我的纪律：不是追求最高收益，而是先活下来 ---------------------</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果只讲一个简单参数，我会这样说：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">卖方不要贪。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">所以我更愿意在中间找平衡。</p>
