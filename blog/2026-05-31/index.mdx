<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你用人民币买一个境内基金。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这个基金拿着监管允许的额度，去海外买资产。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">02场外 QDII：适合懒人定投</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">场外 QDII，就是你在支付宝、银行 App、基金平台里买的基金。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">它适合不想盯盘的人。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你每个月投一点，长期慢慢买。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">不需要研究盘口。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">不需要每天看溢价。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">也不需要判断今天几点买更便宜。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">对大多数普通人来说，这已经够用了。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">它的优点是简单。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">买入门槛低。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">操作熟悉。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">适合长期定投。</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果你只是想长期配置一点标普或纳指，场外 QDII 是最省心的选择。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">尤其是资金量不大的人，不要一开始就把事情搞复杂。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">先开始。先坚持。先别乱追。这比研究一堆高级通道更重要。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">03银行渠道为什么要重点看？</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">很多人买基金只看支付宝。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但如果你真的想长期买 QDII，银行渠道也要看。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">原因很简单：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;"><strong>QDII 有额度。</strong></p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">不同平台的额度不一定一样。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">支付宝限购，不代表全市场都限购。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">有些银行还能买。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">有些券商还能买。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">有些基金公司直销平台也还能买。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">尤其是资金量稍微大一点的人，只靠支付宝会很被动。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">银行渠道还有一个好处：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">有些产品和服务更适合大额资金。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">比如申购额度、赎回安排、账户资金管理和客服处理。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">当然，银行也不一定便宜。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你还是要看费率。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">不要因为是银行，就默认更好。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">更不要被客户经理一顿介绍，就买了自己看不懂的产品。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">看三个东西就够了：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">跟踪什么指数。 总费率多少。 现在还能买多少。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果这三个问题都答不清楚，就先别买。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">04场内 QDII ETF：适合会看溢价的人</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">场内 QDII ETF，就是你用 A 股账户买的跨境 ETF。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">比如纳指 ETF、标普 ETF、海外科技 ETF。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">它的优点是灵活。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">盘中可以买卖。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">有些还能 T+0。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你可以自己控制买入价格和节奏。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但它最大的坑，是溢价。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">什么叫溢价？</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">简单说：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这个 ETF 本来值 100 元。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但因为太多人想买，被炒到 105 元。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">你以为自己在买纳指。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">其实你还多付了 5 元上车费。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这就是溢价。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">场内 ETF 最容易在美股大涨、人民币投资者情绪很热的时候出现高溢价。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这个时候不要冲动。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">因为你买进去以后，就算纳指没跌，只要溢价回落，你也可能亏钱。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">上图我说得比较激进，对于大部分人保守一点的是：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">溢价低于 3%，可以关注。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">溢价高于 5%，最好停手。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">不是说高溢价一定不能买，我的前提是你要保持定投的节奏用时间来降低波动率。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">05香港开银行卡</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">那我去香港开银行卡，是不是就能直接买美股了？</p>

<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果一样东西真的能赚钱，那么你连开户都会变得异常困难。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">开香港银行卡，本身不是问题。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">很多人去香港旅游、消费、留学、工作、收款，都可能需要香港银行账户。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但“有香港银行卡”和“适合直接投资美股”，不是一回事。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">尤其是最近，香港银行和券商对内地客户的审核明显更严。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">开了港卡以后你可以通过银行券商高手续费买美股，互联网券商在 2023 之后基本和大陆人已经无缘了。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">重点不是你能不能开卡。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">重点是你后面拿什么钱去投资。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">钱从哪里来？</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">是不是境外合法收入？</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">是不是境外资产回款？</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">是不是能解释清楚用途？</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果你的钱本来就在大陆，是人民币工资和境内存款，那就不要简单理解成：</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">我开了香港卡，就能绕过去买美股。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这里面涉及外汇用途、资金来源、投资账户审核和后续税务问题。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;"><strong>香港银行卡是账户工具，不是魔法通道。</strong></p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果你只是想旅游消费，或者有真实跨境生活需求，香港银行卡有用。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但如果你只是想把大陆资金搬出去买美股，那就要非常谨慎。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">06全权益 RWA</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">这个我就不做过多解释了，懂的人自然会操作，不懂的人自己搜索了解一下即可。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">07不同大陆投资者怎么选？</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果你资金在 10 万以内。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">优先场外 QDII。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">支付宝、银行 App、基金平台都可以。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">重点是先开始定投，别折腾太多。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果你资金在 10 万到 50 万。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">可以用场外 QDII 做底仓。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">再用一部分资金看场内 ETF。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">但前提是你会看溢价。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">不会看，就不要乱买场内。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">如果你资金在 50 万以上。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">可以多渠道比较。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">支付宝、银行、券商、基金直销都看看。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">场外基金看限购和费率。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">场内 ETF 看成交量和溢价。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">资金越大，越要减少无意义损耗。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">08最后结论</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">大陆人买标普和纳指，主路径就是 QDII。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">支付宝和银行，是最容易开始的入口。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">场外 QDII 适合长期定投。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">场内 QDII ETF 适合会看溢价的人。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">香港银行卡可以开，但不要神化。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">它是账户工具，不是绕开规则的魔法通道。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">全权益 RWA 可以关注，但现在还不是普通大陆人买美股指数的主路径。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">普通人最重要的，不是找到所谓最高收益通道。</p>
<p style="margin:0 0 16px;color:#454550;font-size:16px;line-height:1.9;">而是找到一条合规、稳定、自己能长期执行的路。</p>
